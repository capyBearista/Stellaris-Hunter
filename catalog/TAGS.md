# Achievement Curation Tags

This document defines preferred tags for `catalog/latest.json` curation blocks. Tags are short, lowercase, hyphenated labels used for filtering and planner grouping.

## Tag Rules

- Use slug format: lowercase letters, digits, and hyphens only.
- Use 2-5 tags per achievement.
- Prefer specific tags over generic tags.
- Include a DLC tag for non-Base game achievements when helpful.
- Reuse existing tags before introducing new ones.
